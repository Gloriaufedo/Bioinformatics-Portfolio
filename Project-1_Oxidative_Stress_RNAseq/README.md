# RNA-seq Differential Gene Expression Analysis (Oxidative Stress)

## Objective
To identify genes and pathways involved in oxidative stress response using RNA-seq data.

## Methods
- Differential expression analysis using DESeq2 (Python implementation)
- Visualization using Matplotlib
- Pathway enrichment using gseapy

## Key Findings
- Significant genes identified under oxidative stress conditions
- Enrichment of oxidative stress-related pathways

## Tools
Python, PyDESeq2, Pandas, GSEApy
